Features
Everything you need to route face tracking into VTube Studio.
📱
iPhone ARKit tracking
50+ ARKit blend shapes over Wi-Fi — full expression range for jaw, eyes, brows, cheeks, tongue and more.
📷
Webcam neural tracking
ONNX models auto-downloaded on first run (~3 MB). Head pose, eye blink, jaw, brows — no iPhone needed.
🔧
Transform config editor
Map any tracking variable to any VTS parameter with math expressions. Built-in evalexpr with math:: functions.
🎨
Built-in template
33-parameter ovrog.json preset covering all major expressions. One click to load and start tracking.
🟢
3-state status indicator
Red = disconnected, yellow = connecting (auth pending), green = connected and streaming data to VTS.
🔄
Auto-reconnect
Automatically reconnects to VTube Studio if the connection drops. Configurable delay.
⚡
Low overhead
Native Rust binary. Minimal CPU/memory usage compared to Electron or Python-based alternatives.
🆓
Free & open source
GPL-3.0 license. No subscription, no telemetry, no limits. Fork of rusty-bridge by ovROG.

Quick Start
Up and running in under 2 minutes.
Open VTube Studio on PC — enable the VTS API in Settings → General → VTube Studio API (port 8001).
Download and launch RBridger for your platform.
Go to the Config tab → click Template to load the built-in 33-parameter config, then Save it.
Go to the Bridge tab → choose iPhone or Webcam → for iPhone, enter your device's local IP address.
Press Connect — VTube Studio will show an auth popup. Accept it. Status turns 🟢 green — you're live!
